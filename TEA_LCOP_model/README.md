# Techno-Economic Analysis (LCOP Framework)

This folder contains documentation supporting the Levelized Cost of Production (LCOP) analysis reported in Figure 2 and Supplementary Figures 6–9 of the manuscript.

Scope:
- Physics-based electrochemical cell model
- Flash equilibrium model for solvent losses
- Levelized cost framework including LTOC, LTVOM, and ROI via Fixed Charge Factor
- Component lifetime assumptions consistent with DOE guidelines
